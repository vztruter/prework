HTML

  • HTML provides structure to a web page
  • The head element contains metadata that aren't visible to the user.
  • The body element contains content that the user sees
  • Any meta tags go inside the head section
  • Child elements are nested inside parent elements
  • Semantic elements are descriptive and help with site accessibility
  • Non-semantic elements are not descriptive but are sometimes necessary
  • Some elements are self-closing, such as img elements
